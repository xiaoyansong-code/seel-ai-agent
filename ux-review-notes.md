# UX Review Notes — User Perspective

## Agents List Page
1. Status Banner 的指标与 Live Agent 卡片内的指标完全重复（847 sessions、91.2%、4.6 CSAT），信息冗余
2. "Expand to more channels" 区域只展示了 Email 和 Social 两张卡片，但 Email Agent 已经在上方列表中存在（Setting Up 状态），逻辑上 Email 不应该再出现在 Expand 区域
3. "New Agent" 按钮在 "Your Agents" 标题右侧，视觉权重很低，容易被忽略
4. Opportunities 区域的 "Activate skills" 链接到 /playbook/skills，但用户可能不理解 Playbook 和 Agent 配置的关系
5. Status Banner 缺少时间范围标注 — "847 conversations this week" 但 resolution rate、CSAT 没有标注是同一周还是全局
6. Setting Up 卡片的 "Step 1 of 3 completed" 没有说明三步分别是什么，缺少整体进度上下文
